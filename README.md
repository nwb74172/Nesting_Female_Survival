This is the code and data necessary to reproduce the work in Bakner et al. 
"Limited influence of nest site vegetation on female wild turkey survival during incubation"
in review at Ecosphere

Nest_Selection_Code_Data - Contains the code for nest site selection

Survival_Model_Code_Data - Contains the encounter history used for both the random effect and slope model. The text files are
the model specifications for JAGs.

WAIC_LOO_Supplementary - Contains code to calculate WAIC and LOO outputs demonstrating the random effect model is no better
in fit or performance. THIS WILL NOT BE INCLUDED IN THE MANUSCRIPT ONLY FOR REVIEW.

Nudds_Board_Supplementary - Contains code that demonstrates the robel pole readings are correlated with the Nudd's board readings
THIS WILL NOT BE INCLUDED IN THE MANUSCRIPT ONLY FOR REVIEW.
